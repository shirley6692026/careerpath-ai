## 核心限制

1. **PDF质量依赖** - PyPDF2/OCR提取质量决定解析上限
2. **密格式PDF** - 所有信息压缩在极少数行内，节标题在行中
3. **非标准节标题** - 如"个人特点"、"优势亮点"等不在关键词表
4. **PDF扫描件** - 需OCR，中文识别准确率影响模块提取

## 当前解析引擎

```
PDF上传 → PyPDF2文本提取 → 文本≥50字? 
  ├─ 是 → 行级解析 + 全文回退
  └─ 否 → Tesseract OCR → 行级解析 + 全文回退
         → 6/7种标准节标题识别 + 同名节合并
```
